# Agentic Weather AI using Amazon Bedrock

This project demonstrates an Agentic AI system using:

- Amazon Bedrock
- Claude Sonnet
- Python
- National Weather Service API
- Streamlit

The agent can:
- Understand location queries
- Generate API calls dynamically
- Fetch real-time weather data
- Convert raw JSON into readable forecasts

Files included:
- weather_agent_cli.py
- weather_agent_web.py
- requirements.txt
